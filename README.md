# SHB-Shopping Zone
সকলের অবগতির জন্য জানানো যাচ্ছে যে ওয়েবসাইটটি এখন ও কোনো পোডাক্ট সেল হচ্ছে নাহ ডেমো হিসাবে ওয়েবসাইটটি খোলা হয়ে। ইনশাআল্লাহ ওয়েবসাইটটি চালু হবে এখন নায় চালু হলে সবাইকে জানিয়ে দেওয়া হবো। আস্থা রাখুন SHB Shopping Zone a  দোয়া ও ভালোবাসা চাই ইনশাআল্লাহ একদিন SHB Shopping Zone আপনাকে কম ও ন্যায মূলে সেবা দিয়ে থাকবে
